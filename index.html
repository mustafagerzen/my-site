Q9 – Unit Conversion (kg ↔ lb)

import java.util.Scanner;
public class Q9_Units {

    System.out.print("Value: ");
    double val = in.nextDouble();
    System.out.print("Type (1=kg→lb, 2=lb→kg): ");

  }
}

Princeton Web Exercises #11 — Standard deviation

📊 Objective: Read 5 numbers, print the average and the sample standard deviation (s = sqrt( Σ(x - mean)^2 / (n - 1) )) in addition to the average.

import java.util.Scanner;

public class StandardDeviation {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);

        double[] x = new double[5];
        System.out.println("Enter 5 numbers:");
        for (int i = 0; i < 5; i++) {
            x[i] = in.nextDouble();
        }

        // mean
        double sum = 0.0;
        for (int i = 0; i < 5; i++) sum += x[i];
        double mean = sum / 5.0;

        // sample standard deviation
        double sq = 0.0;
        for (int i = 0; i < 5; i++) {
            double d = x[i] - mean;
            sq += d * d;
        }
        double s = Math.sqrt(sq / 4.0); // n-1 = 4 for sample SD

        System.out.println("Average: " + mean);
        System.out.println("Sample standard deviation: " + s);
    }
}

Princeton Web Exercises #13 — LeapYear arguments (what happens?)

📅 Objective: Understand how a typical LeapYear program that expects a single integer command-line argument behaves with different invocations.

Reference LeapYear program (typical):

public class LeapYear {
    public static void main(String[] args) {
        int year = Integer.parseInt(args[0]); // expects exactly one integer
        boolean isLeap = (year % 4 == 0) && (year % 100 != 0 || year % 400 == 0);
        System.out.println(isLeap);
    }
}

What happens for each run:

  • java LeapYear → ArrayIndexOutOfBoundsException (no args[0]).
  • java LeapYear 1975.5 → NumberFormatException (not an integer).
  • java LeapYear -1975 → Program runs; prints false with the usual rule above (negative years still compute a boolean).
  • java LeapYear 1975 1976 1977 → Program uses only args[0] (=1975); extra args are ignored; prints false.

Note: Many safer versions first check args.length == 1 and print a usage message instead of throwing exceptions.

Princeton Web Exercises #14 — Compiler behavior with string in arithmetic

🧩 Objective: What happens for int a = 27 * "three"; ? Explain and show the correct approach.

Answer: It does not compile. In Java, arithmetic operators require numeric operands. A String cannot be multiplied by an int, so the compiler reports a type error (bad operand types for *).

// Won't compile:
// int a = 27 * "three";   // ERROR: String cannot be used in arithmetic

// One correct approach is to convert a numeric string to an int:
import java.util.Scanner;

public class MultiplyWordFix {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);

        // Example: read a number as text, then parse it
        System.out.print("Enter a whole number as text (e.g., 3): ");
        String s = in.next();

        int value = Integer.parseInt(s); // convert String to int
        int a = 27 * value;

        System.out.println("27 * " + value + " = " + a);
    }
}

Princeton Creative Exercises #21

🌠 Objective: Convert equatorial coordinates (declination δ, hour angle H, latitude φ) to horizontal coordinates (Altitude, Azimuth). (Use radians.)

import java.util.Scanner;

public class MyProgram {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.println("Enter declination δ (radians): ");
        double dec = scanner.nextDouble();
        System.out.println("Enter hour angle H (radians): ");
        double angle = scanner.nextDouble();
        System.out.println("Enter latitude φ (radians): ");
        double lat = scanner.nextDouble();

        double altitude = Math.asin(Math.sin(lat) * Math.sin(dec) +
                                    Math.cos(lat) * Math.cos(dec) * Math.cos(angle));

        double azimuth  = Math.acos((Math.cos(lat) * Math.sin(dec) -
                                     Math.sin(lat) * Math.cos(dec) * Math.cos(angle)) /
                                     Math.cos(altitude));

        System.out.println("Position in horizontal coordinates:");
        System.out.print(altitude + ", ");
        System.out.print(azimuth);
    }
}

Princeton Web Exercises #30 — Car loan payments

🚗 Objective: Read loan amount P (dollars), term Y (years), and annual interest rate R (percent). Compute the fixed monthly payment using monthly compounding:
payment = (P * r) / (1 - (1 + r)^(-n)), where n = 12 * Y and r = (R / 100) / 12.

import java.util.Scanner;

public class CarLoan {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);

        System.out.print("Enter loan amount P (dollars): ");
        double P = in.nextDouble();

        System.out.print("Enter term Y (years): ");
        int Y = in.nextInt();

        System.out.print("Enter annual interest rate R (percent): ");
        double R = in.nextDouble();

        int n = 12 * Y;                 // total number of monthly payments
        double r = (R / 100.0) / 12.0;  // monthly interest rate

        double payment = (P * r) / (1.0 - Math.pow(1.0 + r, -n));

        // Also show a rounded-to-cents version (optional)
        double paymentRounded = Math.round(payment * 100.0) / 100.0;

        System.out.println("Monthly payment (raw): " + payment);
        System.out.println("Monthly payment (rounded to cents): " + paymentRounded);
    }
}
